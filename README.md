## Learning Outcomes of the Experiment
After completing this experiment, the student will be able to:

1. Understand the concept of Single Page Applications (SPA)
2. Create a React application using Vite
3. Implement client-side routing using React Router
4. Build multiple components such as Home, About, and Contact pages
5. Configure and deploy a React application to GitHub Pages
6. Deploy a React application to Netlify
7. Understand basic project structure and build process in React

## Technologies Used
- React
- Vite
- React Router DOM
- GitHub
- Netlify
